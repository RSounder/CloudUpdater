To install 'Simatic IoT 2000 Starter App' on your device,
unzip the zip file into any folder and then click on the executable file.

1: This version of the application deals with only sending, bulk sending, recieving and 
manipulating data from a particular channel of the Thingspeak cloud to and from Simatic IoT 2040 using MQTT 
2: Future versions with cpabilities of offline data transfer such as Modbus, Profinet, Ethernet, USB with Simatic IoT 2040 will be 
released shortly.
